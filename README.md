# Montreal Metro App Website

Official website for the STM Metro App - a native iOS application for navigating Montreal's metro system.

## About

This repository hosts the GitHub Pages website for the [STM Metro App](https://github.com/adamrais/stmMetroApp).

## View Website

Visit the live site at: [https://adamrais.github.io/MontrealMetroApp-website/](https://adamrais.github.io/MontrealMetroApp-website/)

## Repository Structure

- `index.html` - Main landing page
- `images/` - App screenshots and assets

## About the App

The STM Metro App provides:
- Real-time metro service status
- Interactive metro maps
- Complete station information
- Schedule and frequency data
- Accessibility information

Built with Swift and SwiftUI for iOS.
